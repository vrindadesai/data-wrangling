# Syllabus

## Overview

Data Wrangling is a three-week introduction to Python and data science as part of [NCSSM's Summer Accelerator program](https://www.ncssm.edu/summer/summer-accelerator). You will build a working knowledge of Python from the ground up during two asynchronous weeks, then apply those skills to real datasets during the in-person week on campus. No prior programming experience is required.

## Communication

Campuswire allows you to post questions to the class both publicly or anonymously. You can also post questions to the instructor alone — please do this in lieu of emailing me.

You can use the forum to ask any kinds of questions. If you are asking questions about your code or the autograder, please follow these guidelines:

- Posts that include code screenshots should be made private.
- Describe your approach and thought process to the code you have written.
- Be specific about what the problem is and how you have tried to fix it.
- Take a look at other related posts. Oftentimes, other students have the same questions as you!

Campuswire is not only for *questions*! If you know the answer to someone else's question, I encourage you to respond. If you find an interesting resource or article, share it with everyone.

## Course Structure

Week 1 and 2 (July 6–19) will be held online and asynchronously. The final week of the course (July 20–24) will be conducted in-person at the NCSSM Durham campus.

During the async weeks, you will write Python programs from scratch — working with data types, variables, expressions, conditional logic, loops, functions, and lists.

During the sync week, we will extend our Python skills into data-focused applications — working with real datasets, external libraries, and machine learning — and finish the week with a machine learning capstone project.

## Asynchronous Weeks

The first two weeks consist of video lessons with a Gradescope concept check after every lesson. There are three projects across the two asynchronous weeks, also submitted through Gradescope. ## Capstone Project

The capstone involves building a **Naive Bayes classifier** for text classification. You'll select a dataset, implement the classifier using probability formulas covered in class, evaluate its accuracy, and present results on the last day of the course.

## Academic Integrity

This course is a learning opportunity, and I expect you to rely on provided resources — office hours, instructor support, and the question-answer forum — to complete your coursework. While you are welcome to use the internet to seek out resources (e.g. YouTube videos on Python basics or data science articles), I discourage tools that do the learning and problem-solving for you.

Generative AI tools including ChatGPT and GitHub Copilot for code generation are **prohibited**. Using AI to complete or debug your assignments will make Week 3 — which relies on content from Weeks 1 and 2 — difficult and unsatisfactory.
